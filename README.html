Usage: ike[d] { file | - | =code } ...
Tokens: [ ] @ name "string" integer --comment
Dictionary:
   if cond s. . a> >a pick over2 over -rot rot
   swap2 swap drop2 drop dup2 dup ~ | & > = neg
   / * - + dec inc lookup redefine def
Example: 5 dup * .
